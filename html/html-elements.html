There are two types of HTML ELEMENTS
1. INLINE ELEMENTS - Elements those fit in single line. And new element is placed next to the older element
2. BLOCK LEVEL ELEMENTS - Elements those fit(occupy) for entire screen width. And new element is placed in the next line to the older element

Div Element

div element is a block level element in HTML and it allows the grouping of other HTML elements.
It acts as a parent element for the inner grouped elements.
The div element can also hold inner div element.
