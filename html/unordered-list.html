Unordered list

Unordered list is represented in bullet format.
Note: Unordered list doesnt change the order/sequence of the list items how they arranged. Rather it just change the representation style of the list items.
To create unordered list we use ul(unordered List) tag. to represent items in the list we use li(list item) tag.
