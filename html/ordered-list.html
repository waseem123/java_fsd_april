Ordered Lists

		Ordered lists are used to represent the list in a specific order. Say- 1,2,3,....N, a,b,c... etc. i,ii,iii.....

		Order lists are mostly used to list out the items in sequence.

		To create ordered list we use ol(Ordered List) tag.
		to represent items in the list we use li(list item) tag.
