CSS DEMO

CSS stands for Cascading Stylesheet. It is used to apply various styles on HTML components. The stylesheets can work on entire page content. It can modify the HTML components based on their dimensions(Height and Width), Color(foreground and background), spacing(margin and padding), borders, shadow(text and box), font properties(font size, font style, font family), animations and mediaqueries. UI and UX

To write the css you have to give property and its associated value in following format :
property : value
